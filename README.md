# day-day-up
good good study,day day up!

### why

1. 实际帮助到自己日常生活。
2. 巩固此前学习开发知识。
3. 体验项目完整开发流程。
4. 推广给身边的朋友。

### how

1. 主要借助AI进行开发。
2. 使用主流的最新技术。
3. 不断推进迭代开发。
4. 第一版只需要PC端，后续开发小程序端。

### 使用的技术栈

+ 后端：
  + SpringBoot 3.x
  + JDK 21
+ 前端：
  + Vue 3 + TypeScript
  + Vite
  + Element Plus
+ 数据库：
  + MySQL 8.0
  + PostgreSQL
  + MyBatis-Plus
  + Redis 7.x
+ 安全认证：
  + Sa-Token
+ 接口文档：
  + Knife4j 4.x
  + Apifox
+ 部署：
  + Docker + Docker Compose
  + Nginx
+ AI：
  + Spring AI

### 实现的功能

1. 记账。
2. ToDo List
3. 成功日记。
4. 记录日常生活。
5. 做学习笔记。
6. 读书笔记。
7. 每日热点新闻/金句。
8. 生成周报、月报。

### 目标

1. 自己从头到尾都不敲一行代码，都让AI生成。
2. 第一版实现功能：ToDo List、记账、成功日记。
